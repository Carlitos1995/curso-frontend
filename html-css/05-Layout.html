O crescente desenvolvimento das tecnologias, leva a necessidade de realizar tarefas de mais simples e informatizada, ou seja, de modo que seja exigido o menor esforço possível por parte das pessoas.

A escola de condução académica, não possui um sistema para a gestão dos seus serviços, todos os indivíduos associados a escola de condução académica: instrutores, alunos, administrador, recepcionistas tem que se deslocar a instituição para dar tratamento a diversos assuntos.
